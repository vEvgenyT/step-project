Свойства для флекс контейнера

По умолчанию (flex-direction: row) флекс элементы внутри контейнера выстраиваются по главной оси(горизонтальная) слева направо. Если применить значение свойства row-reverse то элементы выстроятся справо налево

Свойство flex-wrap имеет три значения: по умолчанию это nowrap, также это wrap и wrap-reverse. Если без переноса контейнеров то они ужимаются вместе с контентом, если установить перенос, то конейнеры будут переноситься на новую строку

Подведем итог по флекс контейнерам:

У флекс контейнера есть свойства: flex-direction, определяющее направление главной оси, со значениями: row и column а также reverse для них

flex-wrap для переноса флекс элементов на новую строку при отсутствии места со значениями nowrap по умолчанию, wrap и wrap-reverse

justify-content для равномерного распределения флекс элементов внутри контейнера со значениями: flex-start, flex-end, center, space-between, space-around

align-items выравнивает элементы по главной оси по вертикали. Имеет значения: center, flex-start, flex-end и stretch

align-content распределяет елементы перпендикулярно главной оси. Имеет значения как и в justify content: flex-start, flex-end, center, space-between, space-around

Свойства флекс элементов

Жадность флекс элемента или flex-grow (насколько элемент больше остальных), flex-shrink коэффициент сжатия элемента (если у элементов задана ширина и они не помещаются в вьюпорт, то они сжимаются, при нулевом значении свойства элемент не будет сжиматься)

align-self переопределяет свойства заданные через align-items для определенного флекс элемента. Доступны свойства: stretch, center, baseline, flex-end, flex-start
